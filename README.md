# 🎬 Movie Ticket Pricing System (Python)

This project is a simple Python-based ticket pricing system that calculates the final cost of a movie ticket based on multiple conditions.

## 🚀 Features
- Age-based eligibility check
- Membership discount system
- Weekend and evening extra charges
- Seat type pricing (Premium, Gold, Regular)
- Final ticket price calculation

## 🧠 Concepts Used
- if, elif, else statements
- Logical operators (and, or)
- Variables and calculations

## 💻 Sample Output
Final price of ticket: 20
